Frequently Asked Questions

Why doesn't the copy application work with local files?

Because the parser only looks for files located on the network, for example \\192.168.....\....

Why does the repair kit (Fetcher.exe) write in the log that it has repaired the application even if I haven't noticed any problems?

Because it repairs the Multiapp.exe application regardless of the situation, and if it doesn't find any other design errors, it will always report that it has repaired mopad.ini and rebuilt the Assembly for the entire application and performs the usual port checks. The final results will be displayed in HEX format and only Tudor Marmureanu can decode them.

Why isn't Launcher.exe working?

Please check if it is copied into the current folder where Multiapp.exe is located.
